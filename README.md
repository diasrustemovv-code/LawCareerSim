# LawCareerSim

A narrative-driven 2D text-based career simulation / visual novel where the player builds a legal career from law student to legendary lawyer, making ethical, professional, and life-defining decisions.

## Core features
- Year-based progression (1 choice = 1 year)
- Stats: Knowledge, Reputation, Stress, Ethics, Money
- Career ladder: Intern → Lawyer → Senior → Partner → Judge/GC/Founder
- Realistic legal cases and moral dilemmas (focus: Kazakhstan context as an option)

Tech (MVP)
- Engine: Unity (2D)
- Language: C#
- UI: Canvas / TextMeshPro
